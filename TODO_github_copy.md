# Copy MindBoost integration-final to https://github.com/nadaajm/mindboost-copy

**All files from the branch are already cloned locally at ./MindBoost-copy**

**To upload to your repo:**

1. `cd MindBoost-copy`

2. `gh auth login`  - Open browser, login as nadaajm, choose GitHub.com, HTTPS, good to go.

3. `git remote add copy-user https://github.com/nadaajm/mindboost-copy.git`

4. `git push copy-user integration-final:main`  - Pushes the branch as main (or change :branchname)
